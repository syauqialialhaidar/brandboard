# Brandboard TVC

A Vue 3 + Vuetify dashboard for analyzing TV ad performance, brand distribution, and competitor trends.

## Features
- General, Internal & External brand analysis dashboards  
- Reusable chart and metric components (MetricCard, PieChartCard, etc.)  
- Built with TypeScript, Chart.js, and Vuetify 3  

## Tech Stack
Vue 3 • Vuetify 3 • Chart.js • TypeScript  

## Setup
```bash
npm install
npm run dev
